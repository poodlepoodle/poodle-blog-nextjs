- Next-mdx-remote-client

## quick start

### visit [poodlepoodle.me](https://poodlepoodle.me) or

```bash
git clone https://github.com/poodlepoodle/poodle-blog-nextjs.git my-blog-name
cd my-blog-name
yarn install
yarn dev
```
